# capstone-ai-portfolio

This repository is a curated portfolio of capstone projects showcasing applied AI systems: RAG chatbots, agentic assistants, and MLOps end-to-end pipelines.

Structure

- `project1_rag_chatbot/` — Retrieval-Augmented Generation chatbot project with deployment demos and evaluation.
- `project2_agentic_assistant/` — Agentic assistant demonstrating tool use, memory, and orchestration.
- `project3_mlops_end_to_end/` — MLOps pipelines, DVC/MLflow examples, CI/CD, and deployment manifests.

- `docs/` — architecture diagrams, case studies, and resume-ready project summaries.
- `assets/` — images, diagrams, and media used across projects.

Quickstart

1. Clone the repository.
2. Open individual project folders for their README and run instructions.

Contributing

- Each project contains its own README with setup and run steps.
- Use issues/PRs to suggest improvements or add project artifacts.
